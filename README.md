# Car Record System (Araba Kayıt Sistemi)

A simple multi-page web project that helps users:
- Register vehicle information (brand, model, year) into a table
- Calculate fuel cost per kilometer based on consumption and fuel price
- View a list of car brands and open official websites

## Pages
- **Home:** Project purpose and navigation
- **Vehicle Registration:** Add vehicles to a table, remove last row or clear all
- **Fuel Calculator:** Calculates TL per km using (L/100km * fuel price) / 100
- **Brands:** Clickable list that opens brand websites in a new tab

## Technologies
- HTML
- CSS
- JavaScript (Vanilla)

## Project Structure
- `index.html` – Home page
- `arackayit.html` – Vehicle registration form and table
- `aracyakit.html` – Fuel calculator
- `arabasiteleri.html` – Brand list & links
- `styles.css` – Common styling

## How to Run
Just open `index.html` in your browser.
